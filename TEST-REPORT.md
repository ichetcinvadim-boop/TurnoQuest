# TurnoQuests 1.1.0 — отчёт проверки

Дата: 2026-07-20.

Автоматическая симуляция: **1058/1058 проверок пройдено**.

Проверены все 100 квестов: порядок, главы, типы, цели, требования, награды, завершение правильным событием и блокировка неправильного. Проверены переходы 1–101, сценарий `set 54` → `reset 34`, сохранение журнала уже выданных наград, офлайн-поиск по UUID/нику, сериализация, аудит, история, ZIP-бэкап, ротации и общесерверная цель.

Отдельно проверено, что конфигурация содержит встроенного NPC и внутренние арены боссов, использует десять внутренних ID боссов и не требует MythicMobs, Citizens или PlaceholderAPI.

Исходники успешно скомпилированы JDK 21 против Paper API 1.21.10 и Vault API. Итоговый JAR не содержит серверные API и сторонние плагины.

Smoke-test Purpur 1.21.10 выполнен только с Vault и провайдером экономики: TurnoQuests 1.1.0 включился без исключений, зарегистрировал команды, распознал встроенных боссов и сохранил офлайн-цепочку прогресса. Наличие клиента Minecraft для визуального клика по GUI и ручного полного боя всё ещё рекомендуется проверить на тестовом сервере перед продакшеном.
