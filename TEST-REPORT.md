# TurnoQuests 1.2.0 — проверка

Версия 1.2.0 полностью удаляет боссов, арены, MythicMobs, Citizens и PlaceholderAPI.

Автоматический набор проверяет:

- ровно 100 квестов и правильный порядок 1–100;
- соответствие каждого квеста своей главе;
- тип, цель, количество, иконку и награду каждого задания;
- выполнение всех 100 заданий правильным событием;
- блокировку неправильного типа и неправильной цели;
- переход на следующий этап и состояние после №100;
- возврат администратора с №54 на №34;
- сохранение журнала уже выданных наград после сброса;
- офлайн-поиск по UUID и последнему нику;
- сериализацию данных, аудит, историю и ZIP-бэкап;
- ежедневную, еженедельную и общесерверную ротацию;
- отсутствие боссовых заданий, арен и посторонних зависимостей;
- встроенного NPC, Vault и награды ExecutableItems.

Сборка выполняется на GitHub Actions с Temurin JDK 21. Успешный workflow автоматически публикует проверенный JAR в папке `release`.
