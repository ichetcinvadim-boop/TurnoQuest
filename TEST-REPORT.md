# TurnoQuests 1.5.1 — проверка

Версия 1.5.1 увеличивает сезонные награды: первая глава выдаёт 500 осколков, награды растут по сложности, а базовая валюта за 100 квестов составляет ровно 300 000.

Автоматический набор проверяет:

- ровно 100 квестов и порядок 1–100;
- соответствие каждого квеста своей главе;
- положительные цели, денежные награды и осколки каждого задания;
- общий сезонный бюджет валюты и осколков;
- постепенный рост наград по главам;
- выполнение всех 100 заданий правильным событием;
- ручную красную/зелёную кнопку и открытие следующего квеста только после полной выдачи;
- отдельные устойчивые очереди денег, бонусов и осколков;
- отсутствие повторной выплаты уже выданной части;
- выдачу PlayerPoints через UUID API, включая офлайн-профили;
- миграцию сохранений 1.4.x без повторной выплаты старых квестов;
- добавление осколков к выполненной, но ещё не забранной старой награде;
- удаление старой очереди предметов;
- возврат администратора на ранний квест без сброса журнала наград;
- сериализацию, атомарное сохранение, аудит, историю и ZIP-бэкап;
- ежедневные, еженедельные и общесерверные цели;
- понятные инструкции, шкалу прогресса, валюту и осколки в GUI;
- контекстный трекер и его скрытие в PvP/ивентах;
- блокировку незрелого урожая, поставленных блоков и искусственных мобов;
- отсутствие ExecutableItems, MythicMobs, Citizens, боссов и арен;
- все маршруты игроковых и административных команд;
- сборку против настоящего Paper API и проверку ABI.

GitHub Actions использует Temurin JDK 21.
